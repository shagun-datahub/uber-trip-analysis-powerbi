# 🚖 Uber Trip Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Uber Trip Analysis Dashboard** built using **Power BI**. The dashboard provides comprehensive insights into booking performance, revenue generation, customer behavior, vehicle performance, and operational trends.

The objective of this project is to transform raw Uber trip data into meaningful business insights through data cleaning, modeling, DAX calculations, and interactive visualizations.
